# Titanic Data Analysis (EDA)

This project performs exploratory data analysis on the Titanic dataset using Pandas and Matplotlib.

Key steps:
- Data cleaning (handling missing values using title-based age imputation)
- Survival rate analysis by gender and passenger class
- Comparison of counts vs rates
- Basic visualizations to support insights

Tools used:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
